# Bootstrap

1. Execute `install.sql` no SQL Editor do Supabase.
2. Adicione o schema `learning` aos schemas expostos pela API.
3. Desative novos cadastros no Supabase Auth; a aplicacao e de uso pessoal.
4. Configure no servidor:

```text
GEMINI_API_KEY
GEMINI_MODEL
```

Não existe papel administrativo obrigatório. Cada usuário cria e estuda suas
próprias trilhas geradas pela IA.
